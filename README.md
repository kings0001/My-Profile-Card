# Testable Profile Card

A responsive, accessible, and test-ready Profile Card built with vanilla HTML, CSS, and JavaScript.

## 🚀 Features
- **Semantic HTML5:** Uses `<article>`, `<figure>`, `<nav>`, and `<section>` for SEO and screen readers.
- **Modern CSS:** Flexbox and Grid layout with a mobile-first responsive design.
- **Dynamic Content:** Displays the current Epoch time in milliseconds, updating every second.
- **Accessibility:** Meets WCAG AA contrast standards and includes keyboard navigation support.
- **Test-Ready:** Includes specific `data-testid` attributes for automated testing compatibility.

## 🛠️ Built With
- HTML5
- CSS3 (Custom Variables & Grid)
- Vanilla JavaScript
